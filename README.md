Summary
This report presents the results of a complete Vulnerability Assessment and Penetration Testing (VAPT) cycle conducted against the internal lab environment as part of Week 2 VAPT training. The assessment followed the Penetration Testing Execution Standard (PTES) and OWASP Web Security Testing Guide (WSTG).
The scope included Metasploitable2, DVWA, and associated virtual machines. The assessment included reconnaissance, vulnerability scanning, exploitation, post-exploitation, and produced this formal report with remediation recommendations
