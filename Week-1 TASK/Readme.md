Overview:

This report presents the findings of a structured Vulnerability Assessment and Penetration Testing (VAPT) exercise conducted within a controlled lab environment. The objective was 
to identify security weaknesses in a simulated network and web application infrastructure,assess associated risks, and recommend appropriate remediation actions. The assessment 
followed the OWASP Web Security Testing Guide and NIST SP 800-115 methodologies.Testing was carried out using industry-standard open-source tools like Nikto and OpenVAS on Metasploitable
